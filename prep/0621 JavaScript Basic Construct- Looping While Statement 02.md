**Instruction**
 1. Using **While** statement, create a loop that displays a series of number from `5` to `0`. Print each number using `console.log`.

**Expected result:**
```JavaScript
5
4
3
2
1
0
```

**Solution**
```JavaScript
let i = 5
while (i >= 0) {
	console.log(`${i}`);
	console.log(i);
	i--;
};
```
